# Position API

> Backend API for Position Bot, which is a brazilian stock analyzer advisor

## Usage

Rename "config/config.env.env" to "config/config.env" and update the values/settings to your own

## Install Dependencies

```
npm install
```

## Run App

```
# Run in dev mode
npm run dev

# Run in production mode
npm start
```
